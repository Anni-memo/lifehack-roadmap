01

Claude Code とは何か

Claude Codeは、Anthropic社が提供するAIコーディングアシスタントだ。ターミナル（コマンドライン）、デスクトップアプリ、Webアプリ、IDE拡張として動作する。

一般的なAIチャットとの違いは明確だ。Claude CodeはあなたのPC上のファイルを直接読み書きできる。コードを書くだけでなく、ファイルを整理し、Webを検索し、APIを叩き、テストを実行する。

プログラマーだけのツールではない。文章を書く人、データを整理する人、情報発信をする人——PCで何かを作るすべての人にとって、最強の相棒になる。

WHAT MAKES IT DIFFERENT
普通のAIチャット: テキストを送り、テキストが返る。それだけ。
Claude Code: ファイルを読み、コードを書き、コマンドを実行し、結果を確認して修正する。あなたの代わりに「手を動かす」AIだ。

03

インストール — 4つの方法

Claude Codeは複数の方法で利用できる。自分に合ったものを選ぼう。

方法1: デスクトップアプリ RECOMMENDED FOR BEGINNERS
最も簡単。claude.com/download からダウンロード。Windows / Mac対応。インストーラーを実行するだけ。
方法2: Webアプリ NO INSTALL
claude.ai/code にアクセスするだけ。インストール不要だが、ローカルファイルへのアクセスは制限される。
方法3: CLI（ターミナル） FOR POWER USERS
Node.jsが必要。ターミナルで以下を実行：
npm install -g @anthropic-ai/claude-code claude
方法4: IDE拡張 FOR DEVELOPERS
VS Code、JetBrains（IntelliJ等）の拡張機能として利用可能。エディタ内でClaude Codeが動く。
04

最初の設定 — CLAUDE.md を書く

Claude Codeをインストールしたら、最初にやるべきことがある。CLAUDE.md を書くことだ。

CLAUDE.mdは、Claude Codeが毎回起動時に読み込む「行動指針書」。ここに書いた内容が、AIの振る舞いを決める。

WHY CLAUDE.MD MATTERS
CLAUDE.mdがないClaude Code = 初日の新入社員。
CLAUDE.mdがあるClaude Code = あなたのことを知っている、3年目の優秀なアシスタント。

ホームディレクトリに ~/.claude/CLAUDE.md というファイルを作成する。最初はシンプルでいい。

# 私のCLAUDE.md ## コミュニケーション - 日本語で応答してください - 簡潔に、要点を先に伝えてください - 保存や変更の前に、必ず確認してください ## 私について - 名前: （あなたの名前） - 職業: （あなたの職業） - 目標: （今取り組んでいること） ## 環境 - OS: Windows 11 - よく使うフォルダ: C:\Users\（ユーザー名）\Documents\

たったこれだけで、Claude Codeの振る舞いは劇的に変わる。日本語で答え、簡潔に話し、勝手に変更せず確認してくる。

これは出発点に過ぎない。使いながら「こうしてほしい」「これはやめて」と気づいたことを追記していく。CLAUDE.mdは生きたドキュメントだ。

05

最初の会話 — 何を頼むか

セットアップが終わったら、実際に使ってみよう。最初の会話で試してほしいことをいくつか紹介する。

試してみよう 1: ファイル整理 EASY
「ダウンロードフォルダの中身を種類別に整理して」
→ Claude Codeがフォルダの中身を確認し、画像・文書・音楽などに分類してくれる。
試してみよう 2: 文章作成 PRACTICAL
「来週の会議用に、このプロジェクトの進捗報告を書いて」
→ プロジェクトフォルダの内容を読み、報告書を自動生成。
試してみよう 3: Webサイト作成 AMBITIOUS
「自己紹介のWebページを作って」
→ HTML/CSSを自動生成し、ブラウザで確認できる形で出力。このサイト自体、Claude Codeで作っている。

重要なのは、完璧な指示を出す必要はないということ。曖昧な依頼でも、Claude Codeは確認しながら進めてくれる。「とりあえずこんな感じで」が通じるのが、従来のプログラミングとの最大の違いだ。

06

育てる — 使うほど賢くなる仕組み

Claude Codeは使い捨てのツールではない。正しく設定すれば、使うほど賢くなる。

その仕組みが「メモリ」と「スキル」だ。

メモリ（memory/） AUTOMATIC
Claude Codeは会話の中で重要な情報を自動的にメモリに保存する。あなたの好み、プロジェクトの方針、過去のフィードバック。次回の会話でこれらを自動的に参照する。

Obsidianなどのノートアプリと連携させると、あなたの全知識にAIがアクセスできるようになる。知識への投資が複利で回り始める。

その1時間の投資が、その後の100時間を変える。
