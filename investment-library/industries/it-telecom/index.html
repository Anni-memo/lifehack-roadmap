WHAT THIS SHELF IS FOR

この棚の使い方

情報通信業は、東証プライム市場でもっとも企業数が多い業種のひとつです。しかし「情報通信業」という括りは広すぎる。通信インフラとプラットフォームとSIerは、稼ぎ方も競争原理もまったく異なります。

だからこそ、このページは一覧ではなく、地図として使うことを意図しています。この業種は何で稼ぎ、どこにモートが宿り、何を問うべきなのか——その地形を整理することが、ここの役割です。

INDUSTRY STRUCTURE

情報通信業は何で稼ぐのか

「情報通信業」という一枚の括りの中に、稼ぎ方が根本的に異なる複数のビジネスモデルが存在します。分析の前に、どのサブセクターを見ているのかを確認することが重要です。

TYPE 01
通信インフラ
回線・基地局・ネットワーク設備に巨額の固定費を先行投資し、月額課金で回収する。規制産業。設備の減価償却が重く、FCF構造の理解が必須。
NTT · KDDI · SoftBank
TYPE 02
SIer・ITサービス
企業向けにシステムを構築・運用する。労働集約的。顧客の業務に深く入り込むほどスイッチングコストが高まり、長期契約につながる。
NTTデータ · 富士通 · NEC
TYPE 03
ソフトウェア・SaaS
一度開発したソフトウェアを繰り返し販売・提供する。限界費用が低く、スケールするほど利益率が改善する。スイッチングコストが競争優位の核心。
オービック · TIS · 弥生
TYPE 04
プラットフォーム
利用者が増えるほど価値が高まるネットワーク効果型のビジネス。勝者総取りになりやすく、一度確立されたポジションは崩れにくい。
リクルート · Mercari · DeNA
QUESTIONS TO ASK

この業種を見るときの問い

業種全体の特性を把握したうえで、個別企業を見るときには以下の問いを持つと、分析が整理されやすくなります。

  • Q01 この企業は何で稼いでいるのか——通信インフラか、ソフトウェアか、プラットフォームか
  • Q02 スイッチングコストはあるか——顧客はなぜ使い続けるのか。乗り換えの障壁は何か
  • Q03 ネットワーク効果はあるか——ユーザーが増えるほど、サービスの価値は高まるか
  • Q04 限界費用はどう変化するか——スケールするほど利益率は上がるか、それとも人員も比例して増えるか
  • Q05 再投資余地はあるか——稼いだキャッシュを高リターンで再投資できる場所が残っているか
  • Q06 高成長と高収益は両立しているか——成長のために収益を犠牲にしていないか
  • Q07 規制・標準化との関係はどうか——事業の持続性が制度に大きく依存していないか
  • Q08 技術の変化にどう対応しているか——クラウド・AIの波は追い風か逆風か
WHERE THE MOAT LIVES

スイッチングコストとネットワーク効果の在り処

情報通信業でモートが宿る場所は、サブセクターによって大きく異なります。

通信インフラ企業のモートは、主に設備投資の規模と規制参入障壁にあります。しかし、それは「競争優位」というより「規制による保護」に近い場合もある。規制が変わればモートは消えます。

SIerやソフトウェア企業のモートは、顧客業務への深い埋め込みにあります。基幹システムを一度入れ替えると、移行コスト・リスク・時間が膨大になる。この「痛み」がスイッチングコストの本質です。

プラットフォーム企業のモートは、ネットワーク効果にあります。しかし、ネットワーク効果は「使われ始める」のが遅い反面、一度確立されると自己強化します。鍵は、そのネットワークが本当に「強化的」かどうかを確認することです。

「スイッチングコストとは、顧客が乗り換えることを選ばない理由の集積である。コストは金銭的なものだけでなく、時間・リスク・習慣・関係性も含む。この業種では、その多くが見えにくいところに埋まっている。」
