ANALYSIS — 02

moat（競争優位）の6タイプ

バフェットは「経済的な堀（Economic Moat）」と呼ぶ。競合他社が何億円を投じても崩せない競争優位性だ。長期投資では、このmoatが時間とともに広がっているかを追い続ける。

🌐
ネットワーク効果
ユーザーが増えるほど価値が上がる。利用者が離れられない仕組みが生まれる。
例：Visa / リクルート / メルカリ
🔒
スイッチングコスト
乗り換えにコスト・時間・リスクがかかりすぎて、顧客が離れられない。
例：キーエンス / サイボウズ / SAP
💎
無形資産・ブランド
ブランド力・特許・ライセンスが価格決定力と顧客ロイヤルティを生む。
例：コカコーラ / 信越化学 / 任天堂
📉
コスト優位
規模・技術・立地などにより、競合より低コストで同じものを提供できる。
例：ニトリ / コストコ / アマゾン
📍
効率的規模
市場規模が限られ、新規参入しても採算が取れないため競合が入ってこない。
例：地域独占インフラ / 特定規格品
🏛️
規制・許認可の壁
法規制・免許・許認可が新規参入を物理的に制限する。
例：銀行・保険・電力・医薬品
