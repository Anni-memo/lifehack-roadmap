財務分析

損益計算書・貸借対照表・キャッシュフロー計算書——
バフェット流で「消費者独占型企業」を見抜く。

参考文献：バフェットの財務諸表を読む力（Mary Buffett / David Clark著）
FINANCIAL ANALYSIS — 01

バフェットが探す企業、避ける企業

バフェットは財務諸表を見るとき、まず企業を2種類に分類する。「消費者独占型」と「コモディティ型」だ。この違いを理解することが、財務分析の出発点になる。

BUFFETT LOVES — バフェットが買う
消費者独占型企業
  • 高い粗利益率（40%以上）が続く
  • 広告や営業に多くを使わなくて良い
  • 設備投資が少なくてすむ
  • 長期負債が少ない、または不要
  • 継続的に自社株買いや配当ができる
  • ROEが安定して高い（15%以上）
  • 例：コカコーラ・キーエンス・信越化学
BUFFETT AVOIDS — バフェットが避ける
コモディティ型企業
  • 価格競争に巻き込まれる
  • 粗利益率が低く（10〜20%）不安定
  • 大規模な設備投資が継続的に必要
  • 長期負債が多く財務が重い
  • 株主還元より設備に資金が消える
  • ROEが低く・不安定
  • 例：鉄鋼・航空・汎用品メーカー
FINANCIAL ANALYSIS — 02

損益計算書（P&L）の読み方

損益計算書で最初に見るのは粗利益率だ。売上から売上原価だけを引いた粗利益の比率が、企業の競争優位性を端的に示す。

📋 損益計算書（P&L）— バフェットが見るポイント
粗利益率
40%以上が理想。消費者独占型企業は競合に価格を下げさせられないため、粗利益率が高く安定している。20%を割る場合は競争優位性が疑わしい。
目安 ≥ 40%
販管費比率
粗利益に対する販管費の比率。30%以下が望ましい。広告・営業に大量のコストをかけなければ売れない企業は、moatが弱い証拠。
目安 ≤ 30%
研究開発費
R&Dに依存しすぎる企業は将来の利益が不確実。バフェットはR&Dが少なくても競争優位を維持できる企業を好む（コカコーラ等）。ただし製薬・半導体は例外。
依存度に注意
純利益率
売上に対する純利益の比率。20%以上が優秀。長期にわたって高い純利益率を維持できる企業は、価格支配力を持っている。
目安 ≥ 20%
減価償却費
設備集約型企業ほど減価償却費が大きく、キャッシュフローと利益の乖離が大きくなる。設備投資が少ない企業ほどFCFが利益に近い。
少ないほど良い
FINANCIAL ANALYSIS — 03

貸借対照表（B/S）の読み方

貸借対照表では、企業がどれだけ借金をせずに事業を運営できているかを見る。消費者独占型企業は自己資本が厚く、長期負債が少ない。

🏦 貸借対照表（B/S）— バフェットが見るポイント
現金・現金同等物
純キャッシュポジション（現金 − 負債）が大きいほど良い。不況時でも自力で乗り切れる財務的安全性を示す。バフェットはバークシャーで常に大量の現金を保有する。
多いほど良い
長期負債
消費者独占型企業は長期負債がほとんどない。年間純利益の3〜4倍以内に収まっているか確認する。それを超える負債は財務リスクのサイン。
純利益×3以内
自己資本比率
総資産に占める自己資本の比率。40〜60%以上が健全。ただし金融・不動産は業種特性上低くなる傾向があるため、同業比較が重要。
目安 ≥ 40%
ROE（自己資本利益率）
自己資本に対して何%の利益を生み出しているか。15%以上が継続していれば優秀。高ROEを借金なしで達成しているかが重要で、過度な借入でROEを嵩上げしている場合は注意。
目安 ≥ 15%
のれん・無形資産
M&Aを繰り返す企業はのれんが積み上がる。のれんが総資産の過半を占める場合は買収依存型の成長を疑う。ブランド・特許等の無形資産が競争優位の源泉になっている場合は別途評価する。
比率に注意
FINANCIAL ANALYSIS — 04

キャッシュフロー計算書の読み方

バフェットがもっとも重視する財務指標のひとつがFCF（フリーキャッシュフロー）だ。「稼いだ現金から設備投資を引いて残る現金」であり、配当・自社株買い・M&Aの原資になる。

💰 キャッシュフロー計算書 — バフェットが見るポイント
営業CF
毎年プラスであることが大前提。純利益より大きい場合は会計上の利益が実態を正確に反映している証拠。逆に純利益 > 営業CFが続く場合は要注意。
毎年プラス必須
設備投資（CAPEX）
消費者独占型企業は設備投資が少なくてすむ。営業CFに対するCAPEXの比率（25%以下が理想）を見る。設備投資が多い企業はFCFが圧迫される。
営業CFの25%以下
FCF（フリーCF）
営業CF − 設備投資 = FCF。FCFが毎年安定して成長しているかが最重要。FCFが増えることで配当増加・自社株買いが可能になり、株主価値が高まる。
増加トレンドが理想
自社株買い
FCFの使途として自社株買いを継続する企業は、株主価値の向上に真剣な経営者であることを示す。長期にわたって自社株を買い続ける企業はバフェットが高く評価する。
継続性を確認
