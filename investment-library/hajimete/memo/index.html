MEMO THEME · moat / 直販モデル
「センサーを売っているのではなく、顧客の問題を解決している」——キーエンスのmoatの本質

キーエンスを分析するとき、多くの人が「営業利益率55%」という数字に驚く。しかし本当に興味深いのは、なぜその利益率が20年以上続いているかだ。

キーエンスは自社製品を製造せず、商品企画・開発・直販に特化している。直販営業マンは「御用聞き」ではなく「コンサルタント」として顧客の生産ラインに入り込み、問題を発見してから製品を提案する。顧客にとっては「キーエンスを使い始めたら離れられない」状態が生まれる。スイッチングコストが極めて高い。

FCFは毎年5000億円超で、純キャッシュは6兆円を超えた（2024年時点）。この現金をM&Aではなく人材・製品開発への再投資に使うという経営方針が、複利効果でモートを広げ続けている。

MEMO THEME · 資本配分 / バフェット視点
バフェットが商社株を買った「本当の理由」——資本配分の規律

2020年、バフェットが日本の5大商社株を買ったとき、日本の投資家は「なぜ今さら商社か」と驚いた。しかしバフェットが見ていたのは業種ではなく、資本配分の規律と株主還元の持続性だった。

三菱商事のPERは当時7〜9倍。純資産より低い価格で買え、配当利回りは5%を超えていた。バフェットがよく言う「素晴らしい企業を公正な価格で」ではなく、この場合は「公正な企業を素晴らしい価格で」に近い局面だったかもしれない。

注目すべきは、バフェット効果で株価が上がった後も三菱商事が自社株買いと増配を継続したことだ。経営者が「株価が上がったから還元は控えよう」とはならず、FCFが許す限り還元し続けた。これが長期投資家を引き付ける理由だ。

MEMO THEME · コスト優位 / 為替ヘッジ戦略
「円高を武器にする」唯一の小売業——ニトリのmoatは為替感応度の逆張り

ニトリのビジネスモデルで最も面白いのは、円高になるほど利益が増えるという構造だ。多くの小売業が為替リスクを「コスト」として捉えているのに対し、ニトリは為替差益を利益に変える設計を意図的に作った。

海外（主にアジア）で製品を製造し、円建てで販売する。円高になれば仕入れコストが下がり、円安になれば逆風になる。しかしニトリは為替ヘッジを巧みに使いながら、長期では「円高局面に攻勢をかける」経営スタイルを確立している。

33期連続増収増益（2024年まで）という記録は、景気の波・為替の波・コロナの波を乗り越えてきた証だ。コスト優位×SPA（製造小売）モデルによる垂直統合が、他の小売には真似しにくい参入障壁を作っている。
