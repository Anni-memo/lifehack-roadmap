## 【発信準備】プロフィール・自己紹介

### X プロフィール
```
X（Twitter）のプロフィール文を書いてください。
・160文字以内
・「誰の」「何の役に立つか」が伝わる
・フォローしたくなる

私について：
・[実践している習慣]
・[発信テーマ]
・[何を目指しているか]
```

### note プロフィール
```
noteのプロフィールページ用の自己紹介を書いてください。
・400〜600文字
・体験談を入れて親しみやすく
・「この人の記事を読みたい」と思わせる
・最後にフォローの呼びかけ
```

### 初めての自己紹介投稿
```
Xに初めて投稿する自己紹介文を書いてください。
・「はじめまして」から始める
・何を発信するか
・なぜ発信しようと思ったか（体験談）
・フォローしてくれた方へのメッセージ
・ハッシュタグ3つ
・140文字 × 2ツイートのスレッドで
```

## 【コンテンツ企画】ネタ出し

### 投稿ネタ30個出し
```
「[テーマ]」についてのX投稿ネタを30個出してください。
・Tips系・体験談系・質問系・名言コメント系をバランスよく
・初心者でもわかる内容で
・具体的で実践しやすいものを優先

テーマ：モーニングメソッド
```

### FAQ形式コンテンツ
```
「[テーマ]」の初心者がよく持つ疑問を10個挙げて、
それぞれへの回答も書いてください。
これをnote記事やFAQページに使いたい。

テーマ：朝活・モーニングルーティン
```

### シリーズ企画を考える
```
「[テーマ]」について、1ヶ月間の連続投稿シリーズを企画してください。
・タイトル：「○○の30日チャレンジ」等
・各日のテーマを一覧で
・最初と最後の投稿文を書いて

テーマ：モーニングメソッド実践
```

## プロンプトを育てるコツ

1. **良い結果が出たプロンプトはここに保存する**
2. **「条件」を具体的に書くほど結果が良くなる**（文字数・対象・目的）
3. **「パターンを複数出して」**と言うと選択肢が増える
4. **最後に「改善点を教えて」**と聞くとさらにブラッシュアップできる
