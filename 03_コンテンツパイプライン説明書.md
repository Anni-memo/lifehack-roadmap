## ステップ1: 録音する

### 使うツール（どれか1つでOK）
| ツール | 費用 | おすすめ度 | 特徴 |
|--------|------|-----------|------|
| **iPhoneのボイスメモ** | 無料 | ★★★ | 一番手軽 |
| **Androidのレコーダー** | 無料 | ★★★ | 標準アプリ |
| **ICレコーダー（実機）** | 5,000〜 | ★★☆ | 音質が良い |
| **ZOOM H1 等** | 15,000〜 | ★★☆ | 高音質 |

### 録音のコツ
- **テーマを1つに絞る**（「今日のモーニングメソッドで気づいたこと」等）
- **1〜3分が最適**（長すぎると後処理が大変）
- **話しながら構造を意識する**（「まず〜、次に〜、最後に〜」）
- **静かな場所で録音**（ノイズが入ると文字起こし精度が落ちる）

### おすすめツール
| ツール | 費用 | 精度 | 使いやすさ |

| **Whisper（OpenAI）** | 無料 | ★★★ | 最高精度・PC必要 |
| **Google ドキュメント音声入力** | 無料 | ★★☆ | リアルタイム変換 |
| **LINE CLOVA Note** | 無料 | ★★☆ | 日本語特化 |

### 推奨ワークフロー（初心者向け）
```
① iPhoneで録音（ボイスメモアプリ）
② Nottaアプリで文字起こし（自動）
③ テキストをコピーしてClaudeに渡す
```

### 推奨ワークフロー（中級者向け）
```
① ICレコーダーで録音
② PCのWhisperで文字起こし（精度が一番高い）
③ Claude Codeに渡して編集
```

## ステップ3: Claudeで編集する

### X投稿用プロンプト（コピペして使う）
```
以下の文字起こしテキストをもとに、X（Twitter）投稿文を3パターン作ってください。

条件：
- 各パターン140文字以内
- 最後に改行して「#ライフハック #モーニングメソッド」をつける
- 読みやすいように適度に改行を入れる
- 読んだ人が「試してみたい」と思う表現にする

【文字起こし】
（ここにテキストを貼り付け）
```

### note記事用プロンプト（コピペして使う）
```
以下の文字起こしをもとに、note記事を作成してください。

条件：
- 読者：30〜50代、自己改善に興味がある人
- タイトル：思わずクリックしたくなる魅力的なもの
- 構成：導入（共感）→ 本文（3つのポイント）→ まとめ（行動促進）
- 文字数：800〜1,200文字
- 文体：親しみやすく、信頼感がある

【文字起こし】
（ここにテキストを貼り付け）
```

### メルマガ用プロンプト（コピペして使う）
```
以下の内容をもとにメルマガを書いてください。

条件：
- 件名：開封率が上がるもの（疑問形や数字を使う）
- 本文：600〜800文字
- 最後に読者への問いかけを入れる
- PS（追伸）もつける

【内容】
（ここにテキストを貼り付け）
```

## ステップ4: X（Twitter）に投稿する

### 投稿のベストプラクティス
| 項目 | 推奨 |
|------|------|
| **投稿頻度** | 1日1〜3回 |
| **最適な時間** | 7:00〜9:00 / 12:00〜13:00 / 21:00〜23:00 |
| **ハッシュタグ** | 2〜3個（多すぎると逆効果） |
| **画像** | あると2〜3倍のリーチ |
| **スレッド** | 長い内容は分割して投稿 |

### 将来的な自動化（中級〜上級）
- **Buffer / Hootsuite**: 予約投稿ツール
- **Zapier / Make**: 他ツールとの連携自動化
- **Claude + API**: 完全自動投稿（プログラミング知識が必要）

## ステップ5: HP・noteに蓄積する

### なぜ蓄積が重要か
- Xはタイムラインでコンテンツがすぐ流れる → 資産にならない
- HPやnoteは**検索で見つかり続ける**（SEO資産）
- 「あの人のコンテンツをまとめて読みたい」ニーズに応えられる

### note運用のコツ
```
① Xに投稿した内容を、週1回まとめてnote記事にする
② 「今週のライフハックまとめ」などシリーズ化する
③ 有料記事を混ぜていく（後に収益化）
④ フォロワーへのメルマガとも連動させる
```

### HP（将来的に）
| フェーズ | 推奨ツール | コスト |
|---------|-----------|--------|
| 開始期 | note | 無料 |
| 成長期 | WordPress（レンタルサーバー） | 月1,000〜2,000円 |
| 本格期 | 独自ドメイン + WordPress | 月2,000〜5,000円 |

## ステップ6: 拡散・育成

### SEO対策（検索で見つけてもらう）
- **キーワードを意識**: 「モーニングルーティン」「朝活」「ライフハック」等
- **定期更新**: 週1回以上のコンテンツ追加
- **内部リンク**: 記事同士をつなげる
- **Googleサーチコンソール**: 無料ツールで検索状況を確認

### SNS拡散
- **他のライフハック系アカウントとつながる**
- **コメント・リプライを丁寧にする**
- **他の人の投稿をリツイート（相互援助）**
- **週1〜月1でスペース（音声配信）をやってみる**

### メールリスト構築（最重要）
- Xフォロワーはアカウント停止で消える可能性
- メールリストは**あなたの永続的な資産**
- 無料ツール：MailChimp、Brevo（旧Sendinblue）
