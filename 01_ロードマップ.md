# ライフハック講師への道 — 完全ロードマップ

作成日: 2026-03-16

---

## 全体像

```
【現在地】
PC + Claude Code + ボイスレコーダー

        ↓ Phase 1（1〜2週間）
【Claude Code 基礎をマスター】

        ↓ Phase 2（2〜4週間）
【AIコンテンツパイプラインを構築】
音声録音 → 文字起こし → AI要約 → X投稿

        ↓ Phase 3（1〜2ヶ月）
【HPを立ち上げ・コンテンツ蓄積】
X + HP で情報発信の仕組みを自動化

        ↓ Phase 4（3〜6ヶ月）
【認知拡大・ファン形成】
SEO・SNS最適化・メールリスト構築

        ↓ Phase 5（6ヶ月〜）
【講師活動スタート】
オンライン講座・セミナー・書籍・コーチング
```

---

## Phase 1: Claude Code 基礎マスター（1〜2週間）

### 目標
- Claude Code でファイルを作る・編集する・実行する感覚を掴む
- 「AIに頼む」思考パターンを身につける

### やること
| 順番 | タスク | 目安時間 |
|------|--------|--------|
| 1 | `02_Claude_Code説明書.md` を読む | 30分 |
| 2 | Claude Code で簡単なテキストファイルを作ってもらう | 15分 |
| 3 | 「今日の気づき」をClaudeに話しかけてまとめてもらう | 毎日10分 |
| 4 | ショートカット・便利コマンドを覚える | 1週間で少しずつ |

### 習得チェック
- [ ] Claudeにファイルを作ってもらえる
- [ ] Claudeに文章をまとめてもらえる
- [ ] エラーが出てもClaudeに聞いて解決できる

---

## Phase 2: AIコンテンツパイプライン構築（2〜4週間）

### 目標
音声録音したものを、自動でX投稿できる形に変換する仕組みを作る

### パイプライン図
```
[ボイスレコーダー]
      ↓ 録音
[音声ファイル (.m4a/.mp3)]
      ↓ 文字起こし（Whisper / iPhone標準機能 / Notta等）
[テキストファイル]
      ↓ Claude / ChatGPTで要約・編集
[X投稿用テキスト / HP記事]
      ↓ 投稿・公開
[X / HP / note等]
```

### やること
| 順番 | タスク |
|------|--------|
| 1 | 文字起こしツールを決める（無料: iPhone標準、有料: Notta/Whisper） |
| 2 | Claude Codeで「まとめプロンプト」を作る |
| 3 | X投稿フォーマットを決める（140文字 or スレッド） |
| 4 | パイプラインを1回手動でやってみる |
| 5 | 自動化できる部分をClaudeと相談して仕組み化する |

### 習得チェック
- [ ] 音声 → テキスト変換ができる
- [ ] Claudeがテキストを投稿用にまとめてくれる
- [ ] X に投稿できる

---

## Phase 3: HP立ち上げ・コンテンツ蓄積（1〜2ヶ月）

### 目標
- 自分のホームページ（HP）を持つ
- X投稿の内容をHPにも蓄積し、検索で見つけてもらえる状態を作る

### HPの選択肢
| ツール | 難易度 | コスト | おすすめ度 |
|--------|--------|--------|-----------|
| **note** | ★☆☆ | 無料〜 | 最初はここから |
| **WordPress** | ★★★ | 月1,000〜 | 本格運用向け |
| **Notion公開** | ★★☆ | 無料〜 | シンプルでOK |
| **独自サイト** | ★★★ | 月数百〜 | 将来的に |

**推奨ルート**: note → 将来的にWordPress or 独自ドメイン

### やること
| 順番 | タスク |
|------|--------|
| 1 | noteアカウント作成・プロフィール設定 |
| 2 | X投稿と同じ内容をnoteにも掲載（コピペでOK） |
| 3 | タグ・カテゴリを統一する（モーニングルーティン、ライフハック等） |
| 4 | 週1本のまとめ記事をClaudeに作ってもらう |
| 5 | Google Search Consoleに登録（SEO対策） |

---

## Phase 4: 認知拡大・ファン形成（3〜6ヶ月）

### 目標
- フォロワーを増やす
- 「この人のコンテンツが好き」というファンを作る

### 戦略
1. **一貫したテーマ**: モーニングメソッド・ライフハックに絞る
2. **毎日投稿**: 短くてもOK。継続が信頼を生む
3. **相互交流**: コメント・リプライに丁寧に返す
4. **メルマガ or LINE公式**: より深い関係をつくる連絡手段
5. **Claudeを活用**: コンテンツ案・投稿文・返信文もAIと協力して作る

### 数値目標（例）
| 時期 | Xフォロワー | note読者 | メルマガ登録 |
|------|-------------|---------|------------|
| 3ヶ月後 | 500人 | 100人 | 50人 |
| 6ヶ月後 | 2,000人 | 500人 | 200人 |
| 1年後 | 5,000人 | 2,000人 | 1,000人 |

---

## Phase 5: 講師活動スタート（6ヶ月〜）

### 目標
ノウハウを「商品」として提供し始める

### 商品の段階（低価格 → 高価格）
```
①  無料コンテンツ（X・note・YouTube）
        ↓
②  有料note / 電子書籍（500〜1,500円）
        ↓
③  オンライン講座（Udemy・自社）（5,000〜30,000円）
        ↓
④  グループコーチング（月額制）（10,000〜50,000円/月）
        ↓
⑤  個別コーチング・セミナー（50,000〜100,000円+）
```

### Claudeの役割
- 講座コンテンツの構成を一緒に考える
- スライド・テキストの原稿を作る
- 受講者の質問への回答文を下書きする
- ランディングページのコピーを書く
