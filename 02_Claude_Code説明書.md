# Claude Code 完全説明書（初心者向け）

作成日: 2026-03-16

---

## Claude Codeとは？

Claude CodeはAI（Claude）をターミナル（黒い画面）から直接操作できるツールです。

**普通のChatGPTとの違い:**
| | ChatGPT（ブラウザ） | Claude Code |
|--|--|--|
| 使い方 | ブラウザで会話 | ターミナルで会話 |
| ファイル操作 | できない | **できる** |
| コード実行 | 限定的 | **できる** |
| 自動化 | 難しい | **得意** |
| 複数ファイル編集 | できない | **できる** |

→ Claude Code は「AIがあなたのPCで直接作業してくれる」イメージです。

---

## 基本的な使い方

### 起動方法
```bash
# ターミナルを開いて以下を入力
claude
```

### 基本的な話しかけ方

**ファイルを作る**
```
「今日のモーニングルーティンについて500文字の記事を書いて、
morning_routine.mdというファイルに保存して」
```

**ファイルを編集する**
```
「morning_routine.mdの最後に、実践のポイントを3つ追加して」
```

**文章をまとめる**
```
「この文章をX（Twitter）用に140文字以内にまとめて」
```

**アイデアを出してもらう**
```
「モーニングメソッドについて、初心者向けの投稿テーマを10個考えて」
```

## よく使うコマンド

| コマンド | 説明 |
|--------|------|
| `claude` | Claude Codeを起動 |
| `/help` | ヘルプを表示 |
| `/clear` | 会話をリセット |
| `/exit` | 終了 |
| `Ctrl + C` | 処理を止める |

## CLAUDE.md の使い方

プロジェクトフォルダに `CLAUDE.md` というファイルを置くと、
Claude Codeが毎回それを読んで「あなたの好み」を覚えてくれます。

**例：CLAUDE.md の内容**
```markdown
# 私について
- ライフハックを発信する講師を目指している
- モーニングメソッドの専門家
- 読者は30〜50代の自己改善に興味がある人

# 文章スタイル
- 親しみやすく、でも信頼感のある文体
- 難しい言葉は使わない
- 具体的な例を入れる
- 最後に行動を促すひと言を入れる

# X投稿のルール
- 140文字以内
- ハッシュタグは#ライフハック #モーニングメソッド を必ず入れる
- 絵文字を適度に使う
```

## よくある質問

**Q: Claude Code は無料？**
A: 月$20（約3,000円）のClaude Proプランが必要です。コスパは非常に高いです。

**Q: 英語がわからなくても使える？**
A: はい。日本語で話しかければ日本語で答えてくれます。

**Q: 失敗したら？**
A: Claudeに「元に戻して」と言えば戻せることが多いです。
   また、重要なファイルは定期的にバックアップしましょう。

**Q: どこまで自動化できる？**
A: 文章作成・ファイル管理・データ整理などはほぼ任せられます。
   X投稿・HP更新も半自動化できます（詳しくは`03_コンテンツパイプライン説明書.md`参照）。
